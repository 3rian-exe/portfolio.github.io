Widget ExhiBiT

Widget ExhiBiT is two 64x64 pixel HUB75 P3 RGB LED matrix panels daisy chained together. They are powered by a Raspberry Pi Zero 2W, but can also be run off of other Raspberry Pi's such as the Zero W, 2, 3, and 4. This project is dependent on Henner Zellers rpi-led-matrix GitHub repository. This is what makes this project possible. This library provides all the atomic functions necessary to write software that's able to use the display without the need to write any lower-level code. The widgets written for this device are primarily written in Python 3 and C++.

The RGB LED matrix panels interface with Pi using a hat by Adafruit. And this device package with the panels is powered by a 5-volt, 15-amp power supply (75 watts). Although it doesn't pull more than approximately 45-50 watts at full power/brightness, it leaves some room for expansion.
